Guía básica: Página web con Python y Flask

  1. Instala Python y Flask
    python3 -m venv venv
    source venv/bin/activate.fish
    pip install flask
  2. Crea tu archivo principal (por ejemplo, web1.py):
    from flask import Flask, render_template
app = Flask(__name__)
@app.route(\"/\")
def home():
    return render_template(\"home.html\")
if __name__ == \"__main__\":
    app.run(debug=True)
  3. Crea una carpeta templates y agrega tus archivos HTML.
  4. Ejecuta tu app:
    python web1.py
  5. Abre http://localhost:5000 en tu navegador.
